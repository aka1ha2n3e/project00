# buildについて
## コンパイル方法
プロジェクトのルート

~~~sh
mkdir {ビルド用ディレクトリ}
cd {ビルド用ディレクトリ}
cmake ..
make
~~~
もしくは

シェルに権限をあたえる．
~~~sh
chmod +x *.sh #権限を与える．二回目以降は不要
~~~

ビルド
~~~sh
./build.sh
~~~

リビルド．ビルドがうまくいかないとき．
~~~sh
./rebuild.sh
~~~

コンパイルチェック．コンパイラー通るかさえ確認できればいい時に．
~~~sh
./compilecheck.sh
~~~

## 実行方法
~~~sh
cd build
./{実行ファイル}
~~~
